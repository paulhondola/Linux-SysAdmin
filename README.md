# Linux Operating System – Administration and Security 

## Overview

This course focuses on deepening fundamental skills in Linux operating systems. It is designed for undergraduate students, specifically in Automation & Computers or Electronics & Telecommunications , as well as those interested in careers in infrastructure, DevOps, or security.

## Logistics

**Delivery:** Hybrid (In-class / Online).

**Duration:** 35 hours ( hours synchronous /  hours asynchronous).

**Level:** Beginner.

**Class Size:** 20–70 students per group.

## Prerequisites

**Hardware:** Personal laptop.
**Knowledge:** Minimal knowledge of using the Linux operating system.
**Specific Skills:** Introduction to the command line and networking concepts.

## Learning Objectives

**Navigate the Linux file system efficiently and locate configuration files.**

**Manage files and directories via the command line (copy, move, delete).**

**Interpret hardware information, identify components, and monitor performance.**

**Configure network interfaces, resolve connectivity issues, and manage DNS/IP addressing.**

**Apply security principles, including user configuration, permissions, and basic system security.**


## Syllabus

The course content is divided into 3 modules containing the following chapters:

1. **Getting Help:** Using man pages, `whatis`, `apropos`, and locating documentation.

2. **Navigating the Filesystem:** Understanding the structure (`/root`, `/home`, etc.) and using navigation commands (`pwd`, `cd`, `ls`).

3. **Managing Files and Directories:** Creating/modifying resources (`mkdir`, `rm`, `mv`) and understanding basic permissions.

4. **Understanding Computer Hardware:** Viewing hardware via Linux (`lshw`, `lsblk`), understanding drivers, and IoT relevance.

5. **Network Configuration:** Using tools like `ifconfig` and `ip`, managing configuration files (`/etc/hosts`), and understanding TCP/IP and DNS.

6. **System and User Security:** Managing users/groups, sudo access, updates/patches, and firewall configuration.
